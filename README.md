# 🥋 Clean Flow Mentor

Bem-vindo ao **Clean Flow Mentor**, a extensão oficial para garantir que o seu código seja lido como uma prosa. 

Desenvolvida para reduzir a carga cognitiva durante o *context switching* entre Backend (PHP) e Frontend (JavaScript), esta ferramenta atua como um mentor em tempo real, ajudando desenvolvedores e estagiários a manterem a consistência visual em todo o ecossistema do projeto.

## ✨ Funcionalidades Principais

* **🕵️‍♂️ Linter de Idiomas do DOM:** Esqueceu o cifrão? A extensão avisa imediatamente se você declarar um elemento da interface (via `querySelector` ou `getElementById`) sem usar o prefixo obrigatório `$`.
* **🐍 Guardião do snake_case:** Adeus, camelCase! O mentor sublinha variáveis e funções que fujam do nosso padrão de nomenclatura `snake_case`, garantindo uniformidade em toda a base de código.
* **📚 Snippets de DocBlocks (Em breve):** Atalhos rápidos para gerar a estrutura perfeita de comentários explicativos (`@param`, `@return`) em qualquer função.

## 🎯 Por que usar?

* **Código como Documentação:** Variáveis claras significam menos tempo tentando entender o que a tela faz.
* **Onboarding Acelerado:** Novos membros da equipe aprendem as convenções enquanto codam, sem precisar decorar PDFs gigantescos.
* **Code Reviews Limpos:** Pare de perder tempo corrigindo nomenclatura nos Pull Requests e foque na regra de negócio!
